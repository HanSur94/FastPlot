SensorTag Y-listener contract (libs/SensorThreshold/SensorTag.m ~line 265):
```
% updateData(X, Y) is THE supported path to mutate X/Y and fire
% notifyListeners_. Direct property assignment (`s.Y = newY`) works for
% MATLAB's PostSet listener ONLY when accessed through the property; a
% local copy (`s_y_ = ...`) does NOT trigger any listener.
function updateData(obj, X, Y)
    obj.X = X(:)';
    obj.Y = Y(:)';
    obj.notifyListeners_();
end
```
